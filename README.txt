# Title
```
TOP WORD APPLICATION
```

# Description
```
Here is a program to find the most frequent words in a text file and the input file can have up to a few
thousand lines. The output will be another text file.
```

# Requirements
```
    * GNU C++ (g++)
    * Linux or Windows
```

# Utilized Libraries & Packages
To run or to development the application you require to have installed packages below:
```
    * iostream
    * string
    * map
    * fstream
    * regex
```

# Sample Execution Command
```
topword "input=f.txt;output=out.txt"
```
